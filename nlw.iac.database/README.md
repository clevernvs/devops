# Terraform

Iniciando o plugin
`````sh
# Para quando for iniciar ou para sempre que alterar algum recurso
terraform init
`````

`````sh
# Verificar se os arquivos estão corretos
terraform fmt
`````

`````sh
# Planificar um plano de execução
terraform plan
`````

`````sh
# Para criar o cluster
terraform apply -auto-deprove --destroy
`````
